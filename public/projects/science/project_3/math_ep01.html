Functions — The Machine Metaphor

Input → Rule → Output

A function $f(x)$ is a rule that assigns exactly one output to each input. Feed it an $x$, it returns $f(x)$. The set of all inputs is the domain. The set of all possible outputs is the range. Graphing $f(x)$ turns the rule into a picture.

Interactive Function Plotter — Select a family

Odd Functions i
$$f(-x) = -f(x)$$

Symmetric about the origin. e.g. $\sin(x)$, $x^3$

Even Functions
$$f(-x) = f(x)$$

Symmetric about the y-axis. e.g. $\cos(x)$, $x^2$

Composition i
$$(f \circ g)(x) = f(g(x))$$

Chain functions together. Output of $g$ becomes input of $f$.

Derivatives

The Instantaneous Rate of Change

The derivative answers: how steep is the curve right here? Take two points, draw a secant line. Slide the points together until they merge — the secant becomes a tangent, its slope the derivative. This limiting process is the heart of calculus.

THE LIMIT DEFINITION

$$f'(x) = \lim_{\Delta x \to 0} \frac{f(x+\Delta x) - f(x)}{\Delta x}$$
Differentiation Rules
Power Rule $\frac{d}{dx}x^n = nx^{n-1}$
Product Rule ${(fg)' = f'g + fg'}$
Chain Rule $\frac{d}{dx}f(g(x)) = f'(g(x)) \cdot g'(x)$
Quotient Rule $\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$
Exponential $\frac{d}{dx}e^x = e^x$

Complex Numbers

The Imaginary Becomes Real

When the quadratic $x^2 + 1 = 0$ demanded a solution, mathematicians invented $i = \sqrt{-1}$. Complex numbers $z = a + bi$ live in a plane — the complex plane — where the real axis and imaginary axis are perpendicular. This simple extension unlocks the most beautiful formula in mathematics: Euler's Identity.

EULER'S IDENTITY — "THE MOST BEAUTIFUL EQUATION"

$$e^{i\pi} + 1 = 0$$

Links e, i, π, 1, and 0 — the five most important constants in mathematics

Euler's Formula in Full
Any complex number can be written in polar form:
$$z = r e^{i\theta} = r(\cos\theta + i\sin\theta)$$
r = |z| — modulus (distance from origin)
θ = arg(z) — argument (angle from real axis)
Multiplication → add angles, multiply moduli
de Moivre: $(re^{i\theta})^n = r^n e^{in\theta}$
Complex Arithmetic
$(a+bi)+(c+di) = (a+c)+(b+d)i$
$(a+bi)(c+di) = (ac-bd)+(ad+bc)i$
$\overline{z} = a-bi$ (conjugate)
$|z|^2 = z \cdot \overline{z} = a^2+b^2$

Congruences & Number Theory

The Arithmetic of Remainders

$a \equiv b \pmod{m}$ means $a$ and $b$ leave the same remainder when divided by $m$. This simple idea — modular arithmetic — underlies modern cryptography, computer science, and the theory of primes. Carl Gauss called number theory "the queen of mathematics."
